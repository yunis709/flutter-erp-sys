# 🛒 نظام المبيعات

## 🎯 مقدمة

نظام المبيعات هو الواجهة الرئيسية للتفاعل مع العملاء، يوفر نقاط بيع سريعة وفعالة مع إدارة شاملة للفواتير والمرتجعات.

---

## 🏛️ هيكل النظام

```mermaid
flowchart TB
    subgraph POS["نقاط البيع"]
        A[واجهة البيع السريع]
        B[مسح الباركود]
        C[إدارة السلة]
    end
    
    subgraph Invoices["الفواتير"]
        D[فواتير البيع]
        E[فواتير آجلة]
        F[فواتير نقدية]
    end
    
    subgraph Returns["المرتجعات"]
        G[مرتجعات كاملة]
        H[مرتجعات جزئية]
        I[مرتجعات تالفة]
    end
    
    subgraph Pricing["التسعير"]
        J[العروض والخصومات]
        K[أنواع الأسعار]
        L[نظام الولاء]
    end
    
    POS --> Invoices --> Returns
    POS --> Pricing
```

### سير عملية البيع

```mermaid
sequenceDiagram
    participant C as العميل
    participant POS as نظام POS
    participant Inv as المخزون
    participant Acc as المحاسبة
    
    C->>POS: تقديم المنتجات
    loop إضافة منتجات
        POS->>Inv: التحقق من المخزون
        Inv-->>POS: المخزون متاح
        POS->>POS: إضافة للسلة
    end
    
    POS->>POS: حساب الإجمالي
    C->>POS: اختيار طريقة الدفع
    
    alt دفع نقدي
        POS->>Acc: قيد: الصندوق (مدين)
    else دفع آجل
        POS->>Acc: قيد: ذمم العملاء (مدين)
    end
    
    POS->>Inv: خصم من المخزون
    POS->>POS: طباعة الفاتورة
    POS-->>C: تسليم الفاتورة
```

## 🔄 المرتجعات

### أنواع المرتجعات

| النوع | الوصف | الإجراء |
|-------|-------|---------|
| **مرتجع كامل** | إرجاع الفاتورة بالكامل | إلغاء الفاتورة + إرجاع للمخزون |
| **مرتجع جزئي** | إرجاع بعض البنود | تعديل الفاتورة + إرجاع للمخزون |
| **مرتجع تالف** | منتج تالف | إرجاع للمخزون التالف |

### سير المرتجع

```mermaid
flowchart LR
    A[اختيار الفاتورة] --> B[تحديد البنود]
    B --> C[سبب الإرجاع]
    C --> D[فحص المنتج]
    D --> E[إرجاع للمخزون]
    E --> F[إنشاء قيد محاسبي]
    F --> G[طباعة إيصال]
```

## 🎁 العروض والخصومات

### أنواع الخصومات

```mermaid
mindmap
  root((الخصومات))
    خصم نسبة
      على منتج
      على فئة
      على الفاتورة
    خصم ثابت
      قيمة ثابتة
      شراء بـ X
    العروض
      اشترِ واحد واحصل على واحد
      عرض مجمع
      خصم موسمي
    الكوبونات
      كود خصم
      قسيمة شراء
```

### جدول العروض

| العرض | النوع | الشروط | الخصم |
|-------|-------|--------|-------|
| عرض الترحيب | نسبة | أول عملية شراء | 10% |
| خصم الموسم | نسبة | مشتريات > 500 | 15% |
| اشترِ 2 احصل على 1 | عرض | منتجات محددة | منتج مجاني |
| كود SAVE20 | كوبون | كود صالح | 20 ريال |
