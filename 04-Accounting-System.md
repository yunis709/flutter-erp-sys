# 💰 نظام المحاسبة

## 🎯 مقدمة

نظام المحاسبة هو النواة الأساسية لـ ERP، يوفر هيكل محاسبي متكامل يتبع المعايير المحاسبية الدولية مع مراعاة المتطلبات المحلية.

---

## 🏛️ هيكل النظام المحاسبي

```mermaid
flowchart TB
    subgraph Chart["شجرة الحسابات"]
        A[الأصول] --> A1[أصول ثابتة]
        A --> A2[أصول متداولة]
        B[الخصوم] --> B1[خصوم طويلة]
        B --> B2[خصوم قصيرة]
        C[حقوق الملكية]
        D[الإيرادات]
        E[المصروفات]
    end
    
    subgraph Operations["العمليات"]
        J[القيود اليومية]
        L[دفتر الأستاذ]
        T[ميزان المراجعة]
    end
    
    subgraph Reports["التقارير"]
        P[قائمة الدخل]
        BS[الميزانية العمومية]
        CF[تدفق نقدي]
    end
    
    Chart --> Operations --> Reports
```

## 📊 شجرة الحسابات

### الهيكل الشجري

```mermaid
graph TD
    A[الحسابات] --> B[الأصول]
    A --> C[الخصوم]
    A --> D[حقوق الملكية]
    A --> E[الإيرادات]
    A --> F[المصروفات]
    
    B --> B1[أصول ثابتة]
    B --> B2[أصول متداولة]
    
    B1 --> B1a[عقارات]
    B1 --> B1b[معدات]
    B1 --> B1c[مركبات]
    
    B2 --> B2a[الصندوق]
    B2 --> B2b[البنك]
    B2 --> B2c[ذمم العملاء]
    B2 --> B2d[المخزون]
    
    C --> C1[خصوم قصيرة]
    C --> C2[خصوم طويلة]
    
    C1 --> C1a[ذمم الموردين]
    C1 --> C1b[قروض قصيرة]
    
    E --> E1[مبيعات]
    E --> E2[إيرادات أخرى]
    
    F --> F1[تكلفة البضاعة]
    F --> F2[مصروفات تشغيلية]
```

## 📝 القيود اليومية

### سير العمل

```mermaid
flowchart LR
    A[إنشاء القيد] --> B[إضافة البنود]
    B --> C[التحقق من التوازن]
    C --> D{موافقة؟}
    D -->|نعم| E[اعتماد القيد]
    D -->|لا| F[إرجاع للتعديل]
    F --> B
    E --> G[تحديث الأرصدة]
```

### القيود المحاسبية الشائعة

#### 1️⃣ بيع نقدي

```
من حـ الصندوق (مدين)          1,150.00
    إلى حـ المبيعات (دائن)              1,000.00
    إلى حـ ضريبة القيمة المضافة (دائن)   150.00
```

#### 2️⃣ بيع آجل

```
من حـ ذمم العملاء (مدين)      1,150.00
    إلى حـ المبيعات (دائن)              1,000.00
    إلى حـ ضريبة القيمة المضافة (دائن)   150.00
```

#### 3️⃣ شراء نقدي

```
من حـ المشتريات (مدين)        1,000.00
من حـ ضريبة القيمة المضافة (مدين) 150.00
    إلى حـ الصندوق (دائن)               1,150.00
```

#### 4️⃣ شراء آجل

```
من حـ المشتريات (مدين)        1,000.00
من حـ ضريبة القيمة المضافة (مدين) 150.00
    إلى حـ ذمم الموردين (دائن)          1,150.00
```
